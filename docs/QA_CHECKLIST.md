# QA Checklist

## Core Regression Checks

- nutritionist sign up and sign in
- patient sign in
- patient creation by nutritionist
- profile creation and edit
- plan generation and edit
- meal photo upload and analysis fallback
- patient dashboard rendering
- weight logging
- grocery list generation

## Before Closing A Sprint

- update `docs/DELIVERY_TRACKER.md`
- update `docs/WORKLOG.md`
- record decisions if scope or architecture changed
- attach test or validation evidence
